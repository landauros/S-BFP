2. Purpose and Scope of Data Collection / æ•°æ®æ”¶é›†ç›®çš„ä¸èŒƒå›´

English:
Our experimental system may automatically collect certain browser- and device-related data to support research analysis. The data collected may include, but is not limited to:

Browser and system information (e.g., User-Agent, browser version, platform, language preferences)

Device display configuration (e.g., screen resolution, pixel ratio, color depth)

Time zone and system locale

Browser feature availability and capability detection results

Fingerprints derived from web APIs such as Canvas, WebGL, and AudioContext

Interaction metadata (e.g., timing, behavioral variance) necessary for evaluating authentication robustness

All collected information is restricted to the minimal set required for scientific analysis. No personal identifiers (e.g., name, email, or IP address) are deliberately collected or linked to fingerprint data.
